my-blog
=======

## For Developper

### develop

```sh
# pkg install
yarn install

# start dev server
yarn develop

# open page
open http://localhost:8000

# gql
open http://localhost:8000/___graphql
```

### deploy

```sh
# build
$ yarn build

# serve
$ yarn start
```

deploy 自体は master に merge されると自動でデプロイされる on [Netfliy](https://www.netlify.com/)
